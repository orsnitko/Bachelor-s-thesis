# Bachelor-s-thesis

У результаті дослідження було проведено огляд наукової літератури, що дозволило обрати моделі машинного навчання для задачі прогнозування. До набору даних Metro Interstate Traffic Volume було додано додаткові ознаки з метою підвищення точності прогнозування. Для моделювання було застосовано алгоритми машинного навчання з використанням дерева рішень та на основі великої кількості дерев (Decision Tree, Random Forest, Cubist). Для кожної з моделей було підібрано гіперпараметри з використанням перехресної перевірки, що дозволило покращити точність прогнозування. Робота також включає візуалізацію отриманих результатів, що ілюструє ефективність прогнозування у порівнянні з фактичними даними.
